Ruby Classes

Classes provide the framework for objects we use in our programs - they are integral to object-oriented programming and are a very powerful part of Ruby. Characters in a video game provide a great example. Rather than build separate objects for a ton of similiar characters, we can create a character type - or class - to base our characters off of. We can give this character class specific attributes (instance variables) or actions (methods) so they can interact with other players in the game. And when we need to create another character, we don't have to code an entirely new object. We can just copy, or initialize, a new character from the class by calling Class.new

To access or create a class, the syntax is simple:

class Character
end

Each class must have an initialize method. This method is used when calling the .new function to create a new object, or instance of the class. Any variables passed with the .new function are available for use in the initialize method and can be turned into instance variables to be used throughout the class. In the initialize method, you can also specificy criteria for a valid object by raising argument errors.

def initialize(name, race, weapon, hometown)
  @name = name
  @race = race
  @weapon = weapon
  @hometown = hometown
  raise ArgumentError.new("Your name is too long!") unless name.length < 50
end

Classes may have as many additional methods as you'd like, both regular and private. Private methods are not available outside the class, and cannot be called explicitly on an instance of an onject. They are available only within the class, to other instance methods. A classes methods are its actions - ways to manipulate its attributes (instance variables) so that the character can be dynamic and interact with other players.

def change_hometown(new_hometown)
  @hometown = new_hometown
end

def attack
  case @weapon
  when "sword"
    puts "You Slash Your Enemy!"
  when "magic"
    puts "You Cast a Bolt of Acid at Your Enemy!"
  when "bow"
    puts "You Fire a Flaming Arrow at Your Enemy!"
  end
end

With this code, a new character can now be created by calling Character.new, and passing through the attributes of the character. Once a character is created, we can now use the Character#change_hometown and Character#attack methods.

char1 = Character.new("The Hope Slayer", "Human", "magic", "Celdeth's Keep")
end

char1.change_hometown("Norwatch Castle")

char1.attack

And voila! You have your brand new character, ready to explore whatever magical world you've created!
